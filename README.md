# LAPORAN PRAKTIKUM ANALISIS DATA
Kelompok Adiyatma Teguh Pratama
Kelas XI RPL 8

1 Tujuan Analisis
Penelitian ini bertujuan untuk mengeksplorasi data penjualan guna mengidentifikasi beberapa poin krusial
Menentukan profil pelanggan dengan tingkat loyalitas dan nilai transaksi tertinggi
Menganalisis tren perkembangan volume penjualan dari waktu ke waktu
Mengukur sejauh mana efektivitas biaya periklanan terhadap peningkatan omzet perusahaan

2 Pembersihan Data
Sebelum melakukan proses analisis dilakukan tahap penyelarasan data untuk menjamin validitas hasil
Standardisasi Format Tanggal Mengubah format tanggal ke dalam bentuk yang seragam guna memfasilitasi visualisasi tren bulanan secara kronologis
Penanganan Data Hilang Mengatasi kekosongan data pada kolom Total Sales dengan mengimplementasikan fungsi kalkulasi Jumlah Barang dikali Harga

3 Hasil dan Pembahasan
Berdasarkan visualisasi dan pemrosesan data yang telah dilakukan ditemukan beberapa poin penting
Tren Penjualan Visualisasi menggunakan grafik garis menunjukkan adanya lonjakan volume penjualan yang signifikan pada bulan bulan akhir tahun Hal ini mengindikasikan perlunya manajemen inventaris yang lebih ketat menjelang periode tersebut
Analisis Pelanggan Utama Melalui pendekatan matriks RFM pelanggan dengan ID 5001 teridentifikasi sebagai konsumen paling potensial dengan frekuensi pembelian sebanyak 4 kali dan total nilai transaksi mencapai Rp 8582000
Analisis Produk Lambat Melalui visualisasi scatter plot ditemukan bahwa beberapa produk dalam kategori Electronics memiliki harga jual yang tinggi namun memiliki tingkat perputaran barang yang rendah sehingga berdampak pada retensi modal kerja

4 Rekomendasi Strategis
Untuk mengoptimalkan profitabilitas dan efisiensi operasional direkomendasikan beberapa langkah berikut
Program Retensi Pelanggan Menyusun program loyalitas khusus seperti pemberian insentif atau membership untuk mempertahankan pelanggan utama seperti ID 5001
Optimasi Anggaran Iklan Berdasarkan hasil analisis regresi linear koefisien iklan menunjukkan nilai positif sebesar 0061 Meskipun pengaruhnya relatif kecil variabel iklan terbukti memberikan kontribusi positif terhadap penjualan sehingga alokasi anggaran iklan layak untuk dipertahankan
Strategi Penjualan Paket Mengatasi penumpukan stok pada produk yang kurang laku dengan menerapkan strategi penjualan paket bersama produk yang memiliki tingkat permintaan tinggi
